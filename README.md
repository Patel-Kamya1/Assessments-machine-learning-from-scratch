# Assessments - Machine Learning From Scratch

This repository contains implementations of core Machine Learning algorithms built completely from scratch using:

- NumPy
- Pandas
- Matplotlib

No machine learning libraries such as scikit-learn, TensorFlow, or PyTorch are used.

---

## Assessments

### Assessment 1 – Naive Bayes & K-Nearest Neighbours
- Gaussian Naive Bayes from scratch  
- KNN using Euclidean / Minkowski distance  
- Data preprocessing and visualization included  

Folder: Assessment-1-NaiveBayes-KNN

---

### Assessment 2 – Linear Regression
- Linear Regression using Gradient Descent  
- Feature normalization  
- R² Score evaluation  
- Loss curve visualization  

Folder: Assessment-2-Linear-Regression

---

### Assessment 3 – K-Means Clustering & PCA
- K-Means clustering from scratch  
- PCA using Eigen decomposition  
- Eigenvalues and eigenvectors  
- Cluster vs Actual Species visualization  

Folder: Assessment-3-KMeans-PCA

---
